## 第一部分：项目背景与技术栈（1 分钟）

> 「Rebol Lab 解决了一个常见问题：学习成果零散、无法系统展示前端能力。

**技术栈**：
- Vue 3 + TypeScript + Vite + Element Plus + SCSS
- Composition API、Pinia 状态管理、Vue Router 懒加载
- SCSS 变量统一管理深色科技风格主题

**为什么选 Vue 而不是 React**？
Vue 3 Composition API 适合快速原型开发，Element Plus 组件库丰富能快速构建深色科技风格 UI，项目规模适合单页应用不需要 SSR。

**我做了什么**：
- 独立完成从需求分析到 UI 设计到全量代码实现
- 展示工程化能力：组件化、类型安全、主题管理
- GitHub Actions 自动化构建和价格数据定时更新」

---

## 第二部分：核心工具（1 分钟）

> 「Rebol Lab 包含 4 个工具，我重点介绍两个：

**测试用例生成器**：
- 根据需求描述生成 7 类测试维度的测试点
- 功能测试、边界测试、异常测试、安全测试、接口测试、性能测试、UI/交互测试
- 这个工具展示了我对测试方法的理解——从测试工程师视角出发

**AI 模型价格雷达**（重点）：
这是本项目最具技术深度的工具，解决的问题是：模型价格变化快，不知道哪个模型性价比高。

**技术亮点**：
1. **数据可靠性设计**：每条数据有 verificationStatus（已核验/需人工核验/可能过期）和 freshnessDays（距上次核验天数），避免把旧数据当成最新价格
2. **成本估算安全**：仅在 priceUnit=元/百万tokens 且 currency=CNY/USD 时提供估算，其他情况显示明确提示，不猜测价格
3. **自动数据管道**：GitHub Actions 定时抓取公开官方价格页面，生成 JSON，前端优先读取生成的 JSON，缺失则回退到手动维护的备用数据

**面试官可能会问**：数据新鲜度是怎么实现的？
答：GitHub Actions 每天 08:00 UTC 执行 `npm run prices:update`，调用 10 个厂商的 parser，每个 parser 用 cheerio 解析公开价格页面，生成 JSON 文件。前端优先读取生成的 JSON，每条数据有 lastCheckedAt 和 verificationStatus，前端展示时计算 freshnessDays。」

## 第三部分：项目限制与后续计划（30 秒）

> 「Rebol Lab 有一些已知限制：

1. **工具生成逻辑为本地规则**：不是真实 AI 生成，是基于模板的本地辅助工具
2. **数据存储在浏览器 localStorage**：刷新页面会保留历史记录，清除浏览器数据会丢失
3. **无后端服务**：纯前端项目，不涉及数据上传或云同步
4. **价格数据为示例数据**：页面多处标注『实际费用以各厂商官方价格页为准』

**后续计划**：
- 如需接入真实 AI API，建议通过后端代理保护 Key，不在前端保存
- 部署到 Vercel，面试时可直接演示

## 面试官 FAQ

**Q: 这个项目和别的求职作品集有什么区别？**

A: Rebol Lab 不只是展示页面，还包含 4 个实际可用的工具：
- 测试用例生成器展示了我对测试方法的理解
- AI 模型价格雷达展示了我对数据可靠性、前端交互和自动化的思考
- Bug 报告生成器 + 项目包装助手展示了我从实际工作痛点出发设计工具的能力

**Q: 为什么价格数据是示例数据？**

A: 因为抓取需要人工维护官方的真实价格数据。我选择明确标注为 `needs_review`，而不是假装是实时准确的价格表。每条数据都有 `verificationStatus` 和 `lastCheckedAt`，避免把旧数据当成最新价格。

**Q: 你遇到了什么困难，怎么解决的？**

A: 比如 AI 模型价格雷达的数据可靠性设计。
问题：价格数据可能随时变动，如何让用户知道数据是否新鲜？
解决：设计了 verificationStatus 和 freshnessDays 机制，每条数据标注上次核验时间，新鲜度分级显示。
问题：自动抓取可能解析失败，如何降级？
解决：前端优先读取生成的 JSON，缺失则回退到手动维护的备用数据；每条数据标注 `sourceNote`，缺失显示「来源待补充」。

**Q: 你怎么保证数据安全？**

A:
1. 所有数据存储在浏览器 localStorage，不涉及服务器上传
2. 不要在前端保存真实 API Key，如需接入 AI API，建议通过后端代理保护
3. AI 模型价格雷达的数据仅供参考，实际费用以各厂商官方价格页和控制台账单为准
