## 已知问题

### 1. AI 模型价格雷达数据为示例数据

**问题**：`src/data/modelPrices.ts` 中的价格数据均为示例数据，标注为 `needs_review`。

**影响**：用户不应将价格作为实际参考。

**缓解措施**：
- 页面多处标注「实际费用以各厂商官方价格页为准」
- 每条数据标注 `verificationStatus` 和 `lastCheckedAt`
- 价格缺失显示「待核验」而非 0

### 2. localStorage 存储限制

**问题**：浏览器 localStorage 有 5MB 限制。

**影响**：大量历史记录可能导致存储溢出。

**缓解措施**：更新记录超过 5 条时自动清除最早的记录（见 `ModelPriceRadar.vue` 中的 `saveSnapshots` 函数）。

### 3. 工具生成逻辑为本地规则

**问题**：测试用例生成器、Bug 报告生成器、项目包装助手均为本地模板，不是真实 AI 生成。

**影响**：生成结果可能不够智能或不符合预期。

**说明**：已在 README 和页面中明确标注「所有生成逻辑基于本地规则，不是真实 AI 生成」。

### 4. 无后端服务

**问题**：纯前端项目，无服务器存储。

**影响**：
- 数据不持久（清除浏览器数据会丢失）
- 无法跨设备同步

**说明**：已在安全说明中标注「无后端服务」。
