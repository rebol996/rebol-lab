# Rebol Lab

> **软件工程应届生 · 前端开发 · 求职作品集**

一个面向软件工程应届生的前端开发求职作品集，展示了 Vue 3 + TypeScript 技术栈与测试思维的综合实践能力。

**注意**：当前版本为本地辅助工具，所有生成逻辑基于本地规则，暂不接入真实 AI API。

---

## 项目定位

Rebol Lab 是一个**本地辅助工具集 + 公开求职作品集**：

- **公开部分**：展示前端开发能力、测试思维、项目经验的求职作品集
- **本地工具**：测试用例生成、Bug 报告生成、项目包装等辅助工具（本地运行，不上传数据）

---

## 适合谁看

- **面试官**：了解候选人的前端技术栈、项目经验和测试思维
- **求职者**：参考项目结构、技术选型和求职作品集展示方式

## 已知限制

1. **工具生成逻辑为本地规则**：不是真实 AI 生成，是基于模板的本地辅助工具
2. **数据存储在浏览器 localStorage**：刷新页面会保留历史记录，清除浏览器数据会丢失
3. **无后端服务**：纯前端项目，不涉及数据上传或云同步
4. **部分工具仍在开发中**：部分标记为"开发中"或"计划中"的功能尚未完成

---

## 安全说明

1. **所有数据存储在本地浏览器**：不涉及服务器上传，不收集用户隐私
2. **localStorage 仅用于保存历史记录**：最大存储 5 条记录，超出后自动清除最早的
3. **无登录或认证机制**：项目为公开作品集，无隐私保护需求
4. **工具描述中的"AI"仅为辅助工具定位**：不代表接入了真实 AI 服务
